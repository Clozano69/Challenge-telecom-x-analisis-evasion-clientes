# Diagnóstico de Evasión de Clientes (Churn) - Análisis Exploratorio

## 📝 Descripción del Proyecto
Este estudio se enfocó en comprender el fenómeno del **Churn** (cancelación de servicios). A través de un análisis exploratorio de datos (EDA), se buscaron patrones de comportamiento que distinguen a los clientes leales de aquellos que deciden abandonar la compañía.

## 🔍 Etapas del Desarrollo
1. **Limpieza y Tratamiento:** Ajuste de formatos, manejo de valores faltantes y normalización de tipos de datos.
2. **Análisis Estadístico:** Cálculo de media, mediana y desviación estándar para entender la distribución del gasto y permanencia.
3. **Identificación de Patrones:** Comparación de variables numéricas entre clientes activos y cancelados.

## 🛠️ Tecnologías Utilizadas
* **Python**
* **Pandas** para procesamiento de datos.
* **Estadística Descriptiva** para la generación de insights iniciales.

## 💡 Insights y Recomendaciones
* Se detectó que el **tiempo de contrato** es el factor más influyente: a menor tiempo de permanencia, mayor es el riesgo de fuga.
* El **gasto total** acumulado funciona como un indicador de compromiso del cliente.
* **Recomendación:** Implementar estrategias de retención enfocadas en el "periodo crítico" de los primeros meses de contrato para reducir la tasa de cancelación temprana.
